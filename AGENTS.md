# AGENTS.md

Operating rules for any AI coding agent in this repo (Claude Code, VS Code Copilot, Cursor, etc.).
Kept thin on purpose so the rules never drift across tools.

1. Read `CLAUDE.md` in full — the canonical operating manual (stack, the 7 agents,
   non-negotiable rules, build stages). Follow it exactly.
2. Read `context/about-osman.md` — who this is for and the writing rules for any drafted text.
3. Read `docs/HERMES_OS_MASTER_SPEC.md` — the single source of truth for what to build.

Do not duplicate or rewrite these into new files. Build in stages. Stop after each stage.
