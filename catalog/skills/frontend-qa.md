Before a Builder app is considered ready, it should pass a practical frontend
QA read.

Check that npm build passes, primary controls are interactive, navigation has
real targets, responsive layouts do not overlap, empty states are intentional,
and any filters, saved states, compare actions, forms, or detail panels work in
client state.

A polished app should feel like a real product surface, not a single hero page.
