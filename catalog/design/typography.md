Use a clear type scale. Headings, labels, body text, metadata, and buttons
should each have distinct roles.

Headings should establish hierarchy instead of relying on decoration. Compact
panels need smaller headings than hero sections.

Avoid tiny unreadable text, especially in dashboards and mobile layouts.
Metadata can be small, but it still needs enough contrast and line height to
scan.

Premium apps need generous line height and careful paragraph width so the
interface feels deliberate instead of crowded.
