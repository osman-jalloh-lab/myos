Clarity comes before decoration. Start with what the user is trying to see,
decide, or do, then style the interface around that intent.

Put data first. Use real labels, meaningful counts, clear status, and useful
examples instead of fake dashboard filler.

Create visual hierarchy with size, weight, spacing, and position. Each section
should have one primary action and a clear reason to exist.

Avoid generic cards that only repeat marketing claims. Every panel should help
the user compare, act, monitor, configure, or recover.
