Build mobile-first and then enhance for wider screens.

There must be no horizontal overflow on mobile. Cards should stack cleanly.
Text should wrap without clipping. Fixed-width elements need responsive limits.

Touch targets should be large enough for fingers, with clear spacing between
interactive controls.

Dense dashboards should adapt into tabs, accordions, single-column sections, or
horizontal-free stacked panels. Do not simply shrink a desktop grid until it
becomes unreadable.
